# 计算机组成原理与历史成因（comp-architecture-history）

目标
- 以“局部详解 + 历史因果”的方式整理计算机各层（从器件到系统）的构成与演化原因。
- 每一章包含：原理简介、关键发明/时间线、核心实现示例、参考资料与教学链接。

结构
- `docs/00-overview.md`：总体架构与分层
- `docs/01-digital-circuits.md`：晶体管、晶闸管、逻辑门、寄存器
- `...`
