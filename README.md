# 计算机组成原理与历史因果（computing-foundations-history）

## 目标
系统化收纳并逐条详解计算机从物理层到应用层的组成原理与关键历史节点，试图把“为什么会这样演化”写清楚——既有工程实现细节，也有历史成因与社会背景。
